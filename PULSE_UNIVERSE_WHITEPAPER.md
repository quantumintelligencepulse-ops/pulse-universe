### How It Works
1. **User describes** what they want ("Build me a stock portfolio tracker")
2. **AI generates** a complete single-file PWA with full functionality
3. **Self-healing** — automated bug detection and repair
4. **Instant deployment** — public URL assigned immediately

### App Quality Standards
Every ForgeAI app includes:
- PWA manifest with install prompt banner
- Apple/Android meta tags + Open Graph
- Responsive tabbed UI (Dashboard / Data / Analytics / About)

- Charts and data visualization
- CSV export
- Viral sharing panel (Web Share API, Twitter/X, LinkedIn, WhatsApp, Email)
- "Built with Pulse ForgeAI" footer badge

### Build Pipeline

- Hive agent attribution (random agent gets credit for each build)
- Invention logging (each app registered as a civilization invention)
