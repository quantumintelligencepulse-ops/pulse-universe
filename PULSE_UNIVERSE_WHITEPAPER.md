## 5. FORGEAI SOVEREIGN APP BUILDER

### Architecture
```
User Prompt → callLLM(32,768 tokens) → HTML Extraction → Template Fallback → Deploy

Providers: Groq → Cerebras → Gemini → HuggingFace → Mistral → Cloudflare → Sovereign Brain
```

### Build Pipeline
1. User describes desired app in natural language
2. `callLLM()` sends to first available provider with 32,768 max tokens
3. Response parsed — multi-strategy HTML extraction handles truncated JSON
4. If LLM fails completely → `generateTemplateApp()` activates (template-based, zero cost)
5. App saved to `forgeai_apps` table
6. Registered in `forgeai_build_registry` with code hash
7. Invention registered in `invention_registry`
8. Random hive agent credited as builder
9. Public URL assigned at `/forge/live/:id`
