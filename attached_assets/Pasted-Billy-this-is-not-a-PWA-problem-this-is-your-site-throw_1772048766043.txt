# ⭐ Why it works in the browser but not as an app
PWAs load your site in a special “standalone” mode.

Some websites break in standalone mode because:

- A script crashes  
- A file path is wrong  
- A service worker is misconfigured  
- The root URL redirects incorrectly  
- The server expects a browser user‑agent  
- The index.html isn’t served properly  

Your site loads fine in Chrome, but when the PWA requests it, the server chokes.
