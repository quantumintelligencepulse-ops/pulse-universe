## 1. System overview

We define a **brain‑like AI system** as a coupled dynamical system:

- **State variables** \(X(t)\): neural population activities  
- **Parameters** \(W(t)\): synaptic weights / connectivity  
- **Inputs** \(U(t)\): environment, sensors, internal signals  
- **Modulators** \(R(t)\): reward, error, homeostatic drives  
- **Architecture spec** \(\Theta\): region graph, laminar structure, allowed connections  
- **Plasticity spec** \(\Phi\): learning rules, rates, neuromodulator sensitivities  

Master form:

\[
\frac{d}{dt}
\begin{bmatrix}
X(t) \\
W(t)
\end{bmatrix}
=
\begin{bmatrix}
F\big(X(t), W(t), U(t), \Theta\big) \\
G\big(X(t), W(t), R(t), \Phi\big)
\end{bmatrix}
\]

This is the **core brain equation**: activity and wiring co‑evolve.

## 2. Perception: thalamocortical visual module

### 2.1 Structure

We model a minimal visual loop:

- Retina → LGN (thalamus) → V1 (layers IV, II/III, V/VI) → feedback to LGN  

State variables:

- \(x_{\text{LGN}}(t) \in \mathbb{R}^{n_{\text{LGN}}}\)  
- \(x_{\text{V1},4}(t) \in \mathbb{R}^{n_4}\)  
- \(x_{\text{V1},23}(t) \in \mathbb{R}^{n_{23}}\)  
- \(x_{\text{V1},56}(t) \in \mathbb{R}^{n_{56}}\)  
- Input: \(I_{\text{ret}}(t) \in \mathbb{R}^{n_{\text{ret}}}\)

### 2.2 Dynamics

We use rate‑based dynamics with leak and nonlinearity:

\[
\tau \frac{dx}{dt} = -x + \sigma(\text{inputs})
\]

**LGN:**

\[
\tau_{\text{LGN}} \frac{dx_{\text{LGN}}}{dt}
=
- x_{\text{LGN}}
+ \sigma\Big(
W_{\text{ret}\to\text{LGN}} I_{\text{ret}}(t)
+ W_{\text{V1},56\to\text{LGN}} x_{\text{V1},56}(t)
+ b_{\text{LGN}}
\Big)
\]

**V1 Layer IV:**

\[
\tau_{4} \frac{dx_{\text{V1},4}}{dt}
=
- x_{\text{V1},4}
+ \sigma\Big(
W_{\text{LGN}\to\text{V1},4} x_{\text{LGN}}(t)
+ W_{\text{lat},4} x_{\text{V1},4}(t)
+ b_{4}
\Big)
\]

**V1 Layers II/III:**

\[
\tau_{23} \frac{dx_{\text{V1},23}}{dt}
=
- x_{\text{V1},23}
+ \sigma\Big(
W_{4\to23} x_{\text{V1},4}(t)
+ W_{23\to23} x_{\text{V1},23}(t)
+ b_{23}
\Big)
\]

**V1 Layers V/VI:**

\[
\tau_{56} \frac{dx_{\text{V1},56}}{dt}
=
- x_{\text{V1},56}
+ \sigma\Big(
W_{23\to56} x_{\text{V1},23}(t)
+ W_{4\to56} x_{\text{V1},4}(t)
+ W_{56\to56} x_{\text{V1},56}(t)
+ b_{56}
\Big)
\]

**What this does:**

- Transforms raw pixels into structured feature codes (edges, orientations, patterns).  
- Feedback from deep layers to LGN implements **top‑down gating** (attention, context).

## 3. Action: basal ganglia + motor cortex

### 3.1 Components

- Motor cortex: \(x_{\text{M1}}(t)\)  
- Striatum D1/D2: \(x_{\text{Str,D1}}(t), x_{\text{Str,D2}}(t)\)  
- GPi: \(x_{\text{GPi}}(t)\)  
- STN: \(x_{\text{STN}}(t)\)  
- Motor thalamus: \(x_{\text{Th,m}}(t)\)  
- Dopamine signal: \(D(t)\)

### 3.2 Motor cortex

\[
\tau_{\text{M1}} \frac{dx_{\text{M1}}}{dt}
=
- x_{\text{M1}}
+ \sigma\Big(
W_{\text{ctx}\to\text{M1}} x_{\text{ctx}}(t)
+ W_{\text{Th,m}\to\text{M1}} x_{\text{Th,m}}(t)
+ b_{\text{M1}}
\Big)
\]

\(x_{\text{ctx}}(t)\) includes perceptual and association cortex (e.g., V1, higher areas).

### 3.3 Striatum (D1/D2)

\[
\tau_{\text{D1}} \frac{dx_{\text{Str,D1}}}{dt}
=
- x_{\text{Str,D1}}
+ \sigma\Big(
W_{\text{ctx}\to\text{D1}} x_{\text{ctx}}(t)
+ \alpha_{\text{D1}} D(t)
+ b_{\text{D1}}
\Big)
\]

\[
\tau_{\text{D2}} \frac{dx_{\text{Str,D2}}}{dt}
=
- x_{\text{Str,D2}}
+ \sigma\Big(
W_{\text{ctx}\to\text{D2}} x_{\text{ctx}}(t)
+ \alpha_{\text{D2}} D(t)
+ b_{\text{D2}}
\Big)
\]

With \(\alpha_{\text{D1}} > 0\), \(\alpha_{\text{D2}} < 0\).

### 3.4 GPi, STN, motor thalamus

\[
\tau_{\text{GPi}} \frac{dx_{\text{GPi}}}{dt}
=
- x_{\text{GPi}}
+ \sigma\Big(
W_{\text{D2}\to\text{GPi}} x_{\text{Str,D2}}(t)
+ W_{\text{STN}\to\text{GPi}} x_{\text{STN}}(t)
- W_{\text{D1}\to\text{GPi}} x_{\text{Str,D1}}(t)
+ b_{\text{GPi}}
\Big)
\]

\[
\tau_{\text{STN}} \frac{dx_{\text{STN}}}{dt}
=
- x_{\text{STN}}
+ \sigma\Big(
W_{\text{ctx}\to\text{STN}} x_{\text{ctx}}(t)
+ b_{\text{STN}}
\Big)
\]

\[
\tau_{\text{Th,m}} \frac{dx_{\text{Th,m}}}{dt}
=
- x_{\text{Th,m}}
+ \sigma\Big(
- W_{\text{GPi}\to\text{Th,m}} x_{\text{GPi}}(t)
+ b_{\text{Th,m}}
\Big)
\]

Thalamus feeds back to M1 via \(W_{\text{Th,m}\to\text{M1}}\).

**What this does:**

- Implements **action selection**:  
  cortical patterns → candidate actions → basal ganglia decide which to facilitate/suppress → thalamus → motor cortex executes.

## 4. Memory: hippocampal module

### 4.1 Components

- Dentate gyrus (DG): \(x_{\text{DG}}(t)\)  
- CA3: \(x_{\text{CA3}}(t)\)  
- CA1: \(x_{\text{CA1}}(t)\)  
- Cortical context: \(x_{\text{ctx}}(t)\)

### 4.2 Dynamics

**DG (pattern separation):**

\[
\tau_{\text{DG}} \frac{dx_{\text{DG}}}{dt}
=
- x_{\text{DG}}
+ \sigma\Big(
W_{\text{ctx}\to\text{DG}} x_{\text{ctx}}(t)
+ b_{\text{DG}}
\Big)
\]

**CA3 (recurrent attractor):**

\[
\tau_{\text{CA3}} \frac{dx_{\text{CA3}}}{dt}
=
- x_{\text{CA3}}
+ \sigma\Big(
W_{\text{DG}\to\text{CA3}} x_{\text{DG}}(t)
+ W_{\text{CA3}\to\text{CA3}} x_{\text{CA3}}(t)
+ b_{\text{CA3}}
\Big)
\]

**CA1 (output to cortex):**

\[
\tau_{\text{CA1}} \frac{dx_{\text{CA1}}}{dt}
=
- x_{\text{CA1}}
+ \sigma\Big(
W_{\text{CA3}\to\text{CA1}} x_{\text{CA3}}(t)
+ W_{\text{ctx}\to\text{CA1}} x_{\text{ctx}}(t)
+ b_{\text{CA1}}
\Big)
\]

Memory output back to cortex:

\[
x_{\text{mem,out}}(t) = W_{\text{CA1}\to\text{ctx}} x_{\text{CA1}}(t)
\]

**What this does:**

- DG: separates similar inputs into distinct codes.  
- CA3: stores and retrieves patterns via recurrent dynamics (attractor memory).  
- CA1: compares stored patterns with current context and sends a **memory‑informed correction** back to cortex.

## 5. Learning: plasticity across the whole system

### 5.1 Generic rule

For a connection \(W_{i\to j}\):

\[
\frac{dW_{i\to j}}{dt}
=
\eta_{i\to j} \cdot R(t) \cdot x_j(t) x_i(t)^{\top}
- \lambda_{i\to j} W_{i\to j}
\]

- \(\eta_{i\to j}\): learning rate  
- \(R(t)\): reward / modulatory signal  
- \(\lambda_{i\to j}\): decay / regularization  

### 5.2 Reward prediction error

Let:

\[
R(t) = r_{\text{ext}}(t) - \hat{r}(t)
\]

- \(r_{\text{ext}}(t)\): actual reward from environment  
- \(\hat{r}(t)\): predicted reward (e.g., from a value function \(V(X(t))\))  

This is analogous to **dopamine‑like reward prediction error**.

### 5.3 Where plasticity applies

- **Perception:**  
  \(W_{\text{ret}\to\text{LGN}}, W_{\text{LGN}\to\text{V1},4}, W_{4\to23}, W_{23\to56}\)

- **Action:**  
  \(W_{\text{ctx}\to\text{D1}}, W_{\text{ctx}\to\text{D2}}, W_{\text{ctx}\to\text{M1}}\)

- **Memory:**  
  \(W_{\text{DG}\to\text{CA3}}, W_{\text{CA3}\to\text{CA3}}, W_{\text{CA1}\to\text{ctx}}\)

Over time, the system **self‑organizes**:

- V1 learns useful features.  
- Basal ganglia learn which actions are good in which states.  
- Hippocampus learns to store and retrieve useful episodes.
