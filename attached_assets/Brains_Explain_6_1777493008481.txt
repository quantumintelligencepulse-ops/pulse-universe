## 1. High‑level architecture

We’ll model a minimal but real‑ish brain loop:

- **Perception:** Retina → Thalamus (LGN) → V1 (layers IV, II/III, V/VI)  
- **Valuation & Action Selection:** Cortex ↔ Basal Ganglia ↔ Thalamus ↔ Motor Cortex  
- **Memory:** Cortex ↔ Hippocampus (CA3, CA1, DG)  
- **Learning:** Hebbian + reward‑modulated plasticity + error‑based updates  

Conceptually:

\[
\text{World} \xrightarrow{\text{Senses}} \text{Perception} \xrightarrow{\text{Valuation}} \text{Action} \xrightarrow{\text{Outcome}} \text{Learning} \xrightarrow{\text{Memory}}
\]

We’ll express all of this as a coupled dynamical system.

## 2. Perception module: Thalamus + V1 (laminar)

We keep the visual mini‑brain, but now treat it as one module in a larger system.

### 2.1 State variables

- \(x_{\text{LGN}}(t) \in \mathbb{R}^{n_{\text{LGN}}}\): thalamic visual relay activity  
- \(x_{\text{V1},4}(t) \in \mathbb{R}^{n_4}\): V1 layer IV  
- \(x_{\text{V1},23}(t) \in \mathbb{R}^{n_{23}}\): V1 layers II/III  
- \(x_{\text{V1},56}(t) \in \mathbb{R}^{n_{56}}\): V1 layers V/VI  

Retinal input:

- \(I_{\text{ret}}(t) \in \mathbb{R}^{n_{\text{ret}}}\)

### 2.2 Dynamics

We use rate‑based dynamics:

\[
\tau \frac{dx}{dt} = -x + \sigma(\text{inputs})
\]

#### LGN

\[
\tau_{\text{LGN}} \frac{dx_{\text{LGN}}}{dt}
=
- x_{\text{LGN}}
+ \sigma\Big(
W_{\text{ret}\to\text{LGN}} I_{\text{ret}}(t)
+ W_{\text{V1},56\to\text{LGN}} x_{\text{V1},56}(t)
+ b_{\text{LGN}}
\Big)
\]

- Feedforward: retina → LGN  
- Feedback: V1 deep layers → LGN  

#### V1 Layer IV

\[
\tau_{4} \frac{dx_{\text{V1},4}}{dt}
=
- x_{\text{V1},4}
+ \sigma\Big(
W_{\text{LGN}\to\text{V1},4} x_{\text{LGN}}(t)
+ W_{\text{lat},4} x_{\text{V1},4}(t)
+ b_{4}
\Big)
\]

#### V1 Layers II/III

\[
\tau_{23} \frac{dx_{\text{V1},23}}{dt}
=
- x_{\text{V1},23}
+ \sigma\Big(
W_{4\to23} x_{\text{V1},4}(t)
+ W_{23\to23} x_{\text{V1},23}(t)
+ b_{23}
\Big)
\]

#### V1 Layers V/VI

\[
\tau_{56} \frac{dx_{\text{V1},56}}{dt}
=
- x_{\text{V1},56}
+ \sigma\Big(
W_{23\to56} x_{\text{V1},23}(t)
+ W_{4\to56} x_{\text{V1},4}(t)
+ W_{56\to56} x_{\text{V1},56}(t)
+ b_{56}
\Big)
\]

**Interpretation:**

- This module turns raw pixels into structured feature codes (edges, orientations, patterns).  
- Feedback from deep layers to LGN implements attention/context gating.

## 3. Action module: Basal ganglia + motor cortex

Now we add the **action selection loop**.

### 3.1 Key components

- **Motor cortex:** \(x_{\text{M1}}(t) \in \mathbb{R}^{n_{\text{M1}}}\)  
- **Striatum (D1/D2):** \(x_{\text{Str,D1}}(t), x_{\text{Str,D2}}(t)\)  
- **Globus pallidus internal (GPi):** \(x_{\text{GPi}}(t)\)  
- **Subthalamic nucleus (STN):** \(x_{\text{STN}}(t)\)  
- **Thalamus (motor relay):** \(x_{\text{Th,m}}(t)\)  

We’ll compress some details but keep the **direct/indirect pathway logic**.

### 3.2 Motor cortex dynamics

Motor cortex receives:

- **State input** from perception (e.g., V1, higher cortex)  
- **Thalamic motor relay** feedback  

\[
\tau_{\text{M1}} \frac{dx_{\text{M1}}}{dt}
=
- x_{\text{M1}}
+ \sigma\Big(
W_{\text{ctx}\to\text{M1}} x_{\text{ctx}}(t)
+ W_{\text{Th,m}\to\text{M1}} x_{\text{Th,m}}(t)
+ b_{\text{M1}}
\Big)
\]

Where \(x_{\text{ctx}}(t)\) can include V1, association cortex, etc.

### 3.3 Striatum (D1/D2) dynamics

Striatum receives:

- **Cortical input** (state + action context)  
- **Dopamine modulation** \(D(t)\) from midbrain (VTA/SN)  

We separate D1 (direct) and D2 (indirect) populations:

\[
\tau_{\text{D1}} \frac{dx_{\text{Str,D1}}}{dt}
=
- x_{\text{Str,D1}}
+ \sigma\Big(
W_{\text{ctx}\to\text{D1}} x_{\text{ctx}}(t)
+ \alpha_{\text{D1}} D(t)
+ b_{\text{D1}}
\Big)
\]

\[
\tau_{\text{D2}} \frac{dx_{\text{Str,D2}}}{dt}
=
- x_{\text{Str,D2}}
+ \sigma\Big(
W_{\text{ctx}\to\text{D2}} x_{\text{ctx}}(t)
+ \alpha_{\text{D2}} D(t)
+ b_{\text{D2}}
\Big)
\]

With \(\alpha_{\text{D1}} > 0\), \(\alpha_{\text{D2}} < 0\) to reflect opposite dopamine effects.

### 3.4 GPi, STN, Thalamus (motor)

We compress the classic loop:

- D1 → inhibits GPi → disinhibits thalamus → facilitates action  
- D2 → via STN → excites GPi → inhibits thalamus → suppresses action  

Simplified:

\[
\tau_{\text{GPi}} \frac{dx_{\text{GPi}}}{dt}
=
- x_{\text{GPi}}
+ \sigma\Big(
W_{\text{D2}\to\text{GPi}} x_{\text{Str,D2}}(t)
+ W_{\text{STN}\to\text{GPi}} x_{\text{STN}}(t)
- W_{\text{D1}\to\text{GPi}} x_{\text{Str,D1}}(t)
+ b_{\text{GPi}}
\Big)
\]

\[
\tau_{\text{STN}} \frac{dx_{\text{STN}}}{dt}
=
- x_{\text{STN}}
+ \sigma\Big(
W_{\text{ctx}\to\text{STN}} x_{\text{ctx}}(t)
+ b_{\text{STN}}
\Big)
\]

\[
\tau_{\text{Th,m}} \frac{dx_{\text{Th,m}}}{dt}
=
- x_{\text{Th,m}}
+ \sigma\Big(
- W_{\text{GPi}\to\text{Th,m}} x_{\text{GPi}}(t)
+ b_{\text{Th,m}}
\Big)
\]

Thalamus then feeds back to motor cortex (already in M1 equation).

**Interpretation:**

- This loop implements **action selection**:  
  patterns of cortical activity → candidate actions → basal ganglia decide which to facilitate/suppress → thalamus → motor cortex executes.

## 4. Memory module: Hippocampus

We add a **hippocampal circuit** to give the system episodic memory and sequence learning.

### 4.1 Components

We’ll model:

- **Dentate gyrus (DG):** pattern separation  
- **CA3:** recurrent attractor network  
- **CA1:** output to cortex  

State variables:

- \(x_{\text{DG}}(t)\), \(x_{\text{CA3}}(t)\), \(x_{\text{CA1}}(t)\)

Cortical input (from perception + context):

- \(x_{\text{ctx}}(t)\)

### 4.2 Dynamics

#### Dentate gyrus (pattern separation)

\[
\tau_{\text{DG}} \frac{dx_{\text{DG}}}{dt}
=
- x_{\text{DG}}
+ \sigma\Big(
W_{\text{ctx}\to\text{DG}} x_{\text{ctx}}(t)
+ b_{\text{DG}}
\Big)
\]

Weights \(W_{\text{ctx}\to\text{DG}}\) are sparse and decorrelating.

#### CA3 (recurrent attractor)

\[
\tau_{\text{CA3}} \frac{dx_{\text{CA3}}}{dt}
=
- x_{\text{CA3}}
+ \sigma\Big(
W_{\text{DG}\to\text{CA3}} x_{\text{DG}}(t)
+ W_{\text{CA3}\to\text{CA3}} x_{\text{CA3}}(t)
+ b_{\text{CA3}}
\Big)
\]

- \(W_{\text{CA3}\to\text{CA3}}\) stores recurrent associations (attractor memory).

#### CA1 (output to cortex)

\[
\tau_{\text{CA1}} \frac{dx_{\text{CA1}}}{dt}
=
- x_{\text{CA1}}
+ \sigma\Big(
W_{\text{CA3}\to\text{CA1}} x_{\text{CA3}}(t)
+ W_{\text{ctx}\to\text{CA1}} x_{\text{ctx}}(t)
+ b_{\text{CA1}}
\Big)
\]

CA1 then projects back to cortex:

\[
x_{\text{mem,out}}(t) = W_{\text{CA1}\to\text{ctx}} x_{\text{CA1}}(t)
\]

This modifies cortical state and thus perception, valuation, and action.

**Interpretation:**

- DG: separates similar inputs into distinct codes  
- CA3: stores and retrieves patterns via recurrent dynamics  
- CA1: compares stored patterns with current context and sends a “memory‑informed” signal back to cortex

## 5. Learning: plasticity across modules

We now define **synaptic update rules** for key pathways.

### 5.1 Generic Hebbian + reward rule

For a connection from population \(i\) to \(j\):

\[
\frac{dW_{i\to j}}{dt}
=
\eta_{i\to j} \cdot R(t) \cdot x_j(t) x_i(t)^{\top}
- \lambda_{i\to j} W_{i\to j}
\]

Where:

- \(\eta_{i\to j}\): learning rate  
- \(R(t)\): global reward / modulatory signal  
- \(\lambda_{i\to j}\): decay / regularization  

We can specialize:

- **Visual learning:** \(W_{\text{LGN}\to\text{V1},4}\), \(W_{4\to23}\), \(W_{23\to56}\)  
- **Action learning:** \(W_{\text{ctx}\to\text{D1}}\), \(W_{\text{ctx}\to\text{D2}}\), \(W_{\text{ctx}\to\text{M1}}\)  
- **Memory learning:** \(W_{\text{CA3}\to\text{CA3}}\), \(W_{\text{DG}\to\text{CA3}}\), \(W_{\text{CA1}\to\text{ctx}}\)  

### 5.2 Dopamine / reward signal

Let:

\[
R(t) = r_{\text{ext}}(t) - \hat{r}(t)
\]

Where:

- \(r_{\text{ext}}(t)\): actual reward from environment  
- \(\hat{r}(t)\): predicted reward (from a value function \(V(X(t))\))  

This is a **reward prediction error**, like dopamine in basal ganglia.

## 6. The full fused system

Collect all state variables into one big vector:

\[
X(t) =
\begin{bmatrix}
x_{\text{LGN}} \\
x_{\text{V1},4} \\
x_{\text{V1},23} \\
x_{\text{V1},56} \\
x_{\text{M1}} \\
x_{\text{Str,D1}} \\
x_{\text{Str,D2}} \\
x_{\text{GPi}} \\
x_{\text{STN}} \\
x_{\text{Th,m}} \\
x_{\text{DG}} \\
x_{\text{CA3}} \\
x_{\text{CA1}}
\end{bmatrix}
\]

And all plastic weights into \(W(t)\).

Then the **master equation** is:

\[
\boxed{
\frac{d}{dt}
\begin{bmatrix}
X(t) \\
W(t)
\end{bmatrix}
=
\begin{bmatrix}
F\big(X(t), W(t), U(t), \Theta\big) \\
G\big(X(t), W(t), R(t), \Phi\big)
\end{bmatrix}
}
\]

Where:

- \(U(t)\): external inputs (retina, body, environment)  
- \(\Theta\): fixed architecture (which modules exist, connectivity pattern)  
- \(\Phi\): plasticity hyperparameters  

This is the **mathematical brain**: perception, action, memory, learning, all in one dynamical system.
