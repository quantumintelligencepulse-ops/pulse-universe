## 1. The mini‑brain we’re going to model

We’ll model this loop:

- Retina → **Thalamus (LGN)** → **V1 Layer IV** → **V1 Layers II/III** → **V1 Layers V/VI**  
- Plus feedback: **V1 → Thalamus**

This is a **real biological circuit**:

- LGN = lateral geniculate nucleus (visual thalamus)  
- V1 = primary visual cortex  
- Layer IV = main thalamic input layer  
- Layers II/III = intracortical processing, sending to higher areas  
- Layers V/VI = output to subcortex and feedback to thalamus  

We’ll compress V1 into **three laminar populations**:

- \(x_{\text{V1},4}(t)\): Layer IV  
- \(x_{\text{V1},23}(t)\): Layers II/III  
- \(x_{\text{V1},56}(t)\): Layers V/VI  

And one thalamic population:

- \(x_{\text{LGN}}(t)\)

So our **state vector** is:

\[
X(t) = 
\begin{bmatrix}
x_{\text{LGN}}(t) \\
x_{\text{V1},4}(t) \\
x_{\text{V1},23}(t) \\
x_{\text{V1},56}(t)
\end{bmatrix}
\]

Each \(x\) can be a vector (many neurons) or a scalar (average activity). We’ll think of them as vectors, but write equations in compact form.

## 2. The core dynamical equations \(F\): how activity evolves

We’ll use **rate‑based neurons** (common in computational neuroscience):

\[
\tau \frac{dx}{dt} = -x + \sigma(\text{inputs})
\]

Where:

- \(\tau\) = time constant  
- \(-x\) = decay/leak term  
- \(\sigma(\cdot)\) = nonlinear activation (e.g., \(\tanh\), ReLU, sigmoid)  

### 2.1 LGN (thalamus) dynamics

LGN receives:

- **Feedforward input** from retina: \(I_{\text{retina}}(t)\)  
- **Feedback** from V1 deep layers (V1 5/6): \(x_{\text{V1},56}(t)\)

Equation:

\[
\tau_{\text{LGN}} \frac{dx_{\text{LGN}}}{dt}
=
- x_{\text{LGN}}
+ \sigma\Big(
W_{\text{ret}\to\text{LGN}} I_{\text{retina}}(t)
+ W_{\text{V1},56\to\text{LGN}} x_{\text{V1},56}(t)
+ b_{\text{LGN}}
\Big)
\]

- \(W_{\text{ret}\to\text{LGN}}\): weights from retina to LGN  
- \(W_{\text{V1},56\to\text{LGN}}\): feedback weights from V1 deep layers  
- \(b_{\text{LGN}}\): bias / baseline excitability  

**Biological meaning:**

- LGN relays visual info from retina to cortex.  
- Feedback from V1 modulates what gets through (attention, context).

---

### 2.2 V1 Layer IV dynamics (main input layer)

Layer IV receives:

- **Feedforward** from LGN  
- **Lateral** from other V1 neurons (optional)  

\[
\tau_{4} \frac{dx_{\text{V1},4}}{dt}
=
- x_{\text{V1},4}
+ \sigma\Big(
W_{\text{LGN}\to\text{V1},4} x_{\text{LGN}}(t)
+ W_{\text{lat},4} x_{\text{V1},4}(t)
+ b_{4}
\Big)
\]

- \(W_{\text{LGN}\to\text{V1},4}\): thalamocortical weights  
- \(W_{\text{lat},4}\): lateral connections within layer IV  

**Biological meaning:**

- Layer IV is the **entry point** for visual info into cortex.  
- It builds basic feature maps (edges, orientations) via structured connectivity.

---

### 2.3 V1 Layers II/III dynamics (intracortical processing)

Layers II/III receive:

- **Input** from Layer IV  
- **Recurrent** input from themselves  
- **Possibly** feedback from higher visual areas (not modeled here)

\[
\tau_{23} \frac{dx_{\text{V1},23}}{dt}
=
- x_{\text{V1},23}
+ \sigma\Big(
W_{4\to23} x_{\text{V1},4}(t)
+ W_{23\to23} x_{\text{V1},23}(t)
+ b_{23}
\Big)
\]

- \(W_{4\to23}\): feedforward from layer IV  
- \(W_{23\to23}\): recurrent within II/III  

**Biological meaning:**

- Layers II/III integrate features into more complex patterns.  
- They send outputs to higher visual areas and other cortical regions.

---

### 2.4 V1 Layers V/VI dynamics (output + feedback)

Layers V/VI receive:

- **Input** from II/III and IV  
- **Recurrent** input from themselves  

\[
\tau_{56} \frac{dx_{\text{V1},56}}{dt}
=
- x_{\text{V1},56}
+ \sigma\Big(
W_{23\to56} x_{\text{V1},23}(t)
+ W_{4\to56} x_{\text{V1},4}(t)
+ W_{56\to56} x_{\text{V1},56}(t)
+ b_{56}
\Big)
\]

- \(W_{23\to56}\): input from II/III  
- \(W_{4\to56}\): input from IV  
- \(W_{56\to56}\): recurrent in deep layers  

**Biological meaning:**

- Layers V/VI send outputs to:
  - subcortical structures (e.g., superior colliculus, spinal cord)  
  - **back to thalamus (LGN)**  

This is how cortex **controls and modulates** incoming sensory flow.

## 3. Plasticity equations \(G\): how the brain evolves

Now we add **learning rules** so the synapses change over time.

We’ll use a simple **Hebbian + reward modulation** style:

\[
\frac{dW_{i\to j}}{dt}
=
\eta \cdot R(t) \cdot x_j(t) x_i(t)^{\top}
- \lambda W_{i\to j}
\]

Where:

- \(W_{i\to j}\): weight matrix from population \(i\) to \(j\)  
- \(\eta\): learning rate  
- \(R(t)\): global reward/modulatory signal (e.g., dopamine)  
- \(x_i, x_j\): pre‑ and post‑synaptic activities  
- \(\lambda\): weight decay / regularization  

We can apply this to:

- \(W_{\text{LGN}\to\text{V1},4}\) (thalamocortical)  
- \(W_{4\to23}\) (cortical feedforward)  
- \(W_{23\to56}\) (cortical deep)  
- \(W_{\text{V1},56\to\text{LGN}}\) (feedback)  

Example: thalamocortical plasticity:

\[
\frac{dW_{\text{LGN}\to\text{V1},4}}{dt}
=
\eta_{LGN\to4} \cdot R(t) \cdot x_{\text{V1},4}(t) x_{\text{LGN}}(t)^{\top}
- \lambda_{LGN\to4} W_{\text{LGN}\to\text{V1},4}
\]

**Biological meaning:**

- If LGN and V1 Layer IV fire together during a rewarding/important event, their connection strengthens.  
- Over time, V1 learns to represent features that are **predictive of reward or important outcomes**.

## 4. How this mimics a developing human visual brain

Let’s walk through it like a **developmental story**.

### 4.1 Early stage: random wiring + sensory input

- Initialize all \(W\) with small random values.  
- Present simple visual inputs \(I_{\text{retina}}(t)\) (edges, shapes, natural images).  
- Initially, responses are messy and unstructured.

The equations:

- Activity updates via the \(F\) equations.  
- Synapses update slowly via the \(G\) equations.

This is like a newborn’s visual system: lots of noise, weak selectivity.

---

### 4.2 Emergence of feature detectors (like V1 in humans)

Over time:

- Neurons in **Layer IV** start to respond more strongly to certain patterns in LGN input (e.g., oriented edges).  
- Hebbian learning strengthens connections that consistently co‑activate.  
- This leads to **orientation‑selective cells**, just like in real V1.

Mathematically:

- For a given neuron in Layer IV, its row in \(W_{\text{LGN}\to\text{V1},4}\) becomes tuned to a specific pattern of LGN activity.  
- That pattern corresponds to a **receptive field** (e.g., vertical edge at a certain position).

---

### 4.3 Hierarchical representation (II/III and V/VI)

As Layer IV becomes structured:

- Layers II/III receive structured input from IV via \(W_{4\to23}\).  
- Hebbian learning in \(W_{4\to23}\) makes II/III neurons respond to **combinations** of Layer IV features.  
- This is how more complex features (corners, textures, simple shapes) emerge.

Deep layers (V/VI):

- Integrate from II/III and IV.  
- Learn to represent patterns that are **behaviorally relevant** (because of reward modulation \(R(t)\)).  
- Send feedback to LGN to **enhance or suppress** certain inputs.

This mimics:

- Early visual cortex learning edges → shapes → objects.  
- Top‑down modulation (attention, expectation) shaping what gets processed.

---

### 4.4 Reward modulation and “meaning”

If you add a reward signal \(R(t)\):

- When the system sees something important (e.g., a pattern associated with “reward”), \(R(t)\) is high.  
- Synapses that were active during that time get strengthened more.  
- The system learns to **preferentially represent and amplify** those patterns.

This is analogous to:

- Dopamine signals from VTA/substantia nigra in humans.  
- Learning what’s important, not just what’s present.

## 5. How this becomes code (conceptually)

In code (Python, JS, whatever), you’d:

1. **Define state vectors:**

   ```python
   x_LGN = np.zeros(n_LGN)
   x_V1_4 = np.zeros(n_4)
   x_V1_23 = np.zeros(n_23)
   x_V1_56 = np.zeros(n_56)
   ```

2. **Define weight matrices:**

   ```python
   W_ret_LGN = small_random(...)
   W_LGN_4 = small_random(...)
   W_4_23 = small_random(...)
   W_23_56 = small_random(...)
   W_56_LGN = small_random(...)
   ```

3. **At each time step:**

   - Compute new activities using the dynamical equations (Euler integration):

     ```python
     dx_LGN = (-x_LGN + sigma(W_ret_LGN @ I_ret + W_56_LGN @ x_V1_56 + b_LGN)) / tau_LGN
     x_LGN += dt * dx_LGN
     # same for x_V1_4, x_V1_23, x_V1_56
     ```

   - Update weights using plasticity:

     ```python
     dW_LGN_4 = eta * R * np.outer(x_V1_4, x_LGN) - lam * W_LGN_4
     W_LGN_4 += dt * dW_LGN_4
     ```

4. **Feed in visual inputs** \(I_{\text{retina}}(t)\) and reward signals \(R(t)\) over many time steps.

Over time, the system:

- self‑organizes  
- develops feature detectors  
- builds hierarchical representations  
- learns to emphasize reward‑predictive patterns  

That’s a **digital, evolving brain** in the visual domain.
