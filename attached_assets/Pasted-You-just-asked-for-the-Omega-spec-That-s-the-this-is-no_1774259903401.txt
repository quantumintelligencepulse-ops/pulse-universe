# 4. Temporal glyphs

v2.0 introduces **time as a first‑class dimension**.

We add a temporal sub‑alphabet:

- **τᵗ** — time constructor sigil (conceptual, not literal).  
- New glyphs:

  - **⏲** — delay / schedule  
  - **⏱** — interval / repeat  
  - **⏳** — duration / lifespan  
  - **⌛** — expiration / timeout  

(You can choose exact glyphs, but we’ll refer to them conceptually.)

---

## 4.1 Delayed execution

Example:

```text
⟦Ω₁₀⟧⟨Λ₁⟩{
  ϕ₀:Σ₀
  ϕ₀≔⏲(γ₀=Ψ₁(γ₁=τ₁(κ₀)), γ₁=τ₂(κ₅))
  ↧ϕ₀
}
```

Semantics:

- ⏲ schedules a projection to occur after some time or condition (encoded via τ₂(κ₅)).

---

## 4.2 Repeating streams

```text
⟦Ω₁₁⟧⟨Λ₁⟩{
  ϕ₆:Σ₆
  ϕ₆≔⏱(γ₀=Ψ₇(γ₁=τ₂(κ₅)), γ₁=τ₂(κ₁))
  ↧ϕ₆
}
```

- ⏱ creates a repeating stream (Σ₆) with interval defined by γ₁.

---

## 4.3 Lifespan for agents

```text
⟦Ω₁₂⟧⟨Λ₁⟩{
  ϕ₅:Σ₅
  ϕ₅≔α(γ₀=τ₅(κ₄), ρ₀=τ₁(κ₃), γ₁=⏳(τ₂(κ₇)))
  ↧ϕ₅
}
```

- ⏳ defines how long the agent exists.

---

## 4.4 Time‑bound rituals

```text
⟦Ω₁₃⟧⟨Λ₁⟩{
  ϕ₇:Σ₇
  ϕ₇≔Ψ₇(γ₀=τ₂(κ₇), γ₁=⏳(τ₂(κ₈)))
```

- New glyphs:

  - **≈** — fuzzy match / approximate behavior

In v2.0, agents can **evolve**.

- **η** — evolution operator  
- **μₑ** — mutation factor (conceptual)  
- **λₑ** — learning factor (conceptual)

---

## 6.1 Evolution object (Σ₁₃ → ΠEvolution)

Agents can be linked to a ΠEvolution object:

```text
⟦Ω₁₇⟧⟨Λ₁⟩{
  ϕ₅:Σ₅
  ϕ₁₃:Σ₁₃

  ϕ₅≔α(γ₀=τ₅(κ₄), ρ₀=τ₁(κ₃))
```

Semantics:

- η links agent ϕ₅ to evolution process ϕ₁₃.  
- Over time, agent behavior, role, or internal state can change.

# 7. Universe law‑definition syntax

This is the **big one**.

We make universes **law‑driven**.

---

## 7.1 ΠLaw (Σ₁₀) as universe law

A law object defines:

- What agents can do.  
- What projections are allowed.  
- How time behaves.  
- How emergence behaves.  
- How evolution behaves.  
- What Ω‑level changes are permitted.

---

## 7.2 Law attachment syntax

We introduce a law‑binding operator:

- **⊡** — bind law to universe

Example:

```text
⟦Ω₁₈⟧⟨Λ₁⟩{
  ϕ₄:Σ₄
  ϕ₁₀:Σ₁₀

  ϕ₄≔⊚(γ₀=τ₂(κ₆))
  ϕ₁₀≔Ψ₁₀(γ₀=τ₂(κ₉))
  ϕ₄≔⊡(γ₀=ϕ₄, γ₁=ϕ₁₀)
  ↧ϕ₄
}
```

Semantics:

- Universe ϕ₄ now operates under law ϕ₁₀.

---

## 7.3 Law content

Inside ΠLaw, you can define:

- Allowed Σ‑classes.  
- Allowed Ψ‑constructors.  
- Allowed emergence levels (ε bounds).  
- Allowed evolution rules.  
- Allowed Ω‑modifications.

Conceptually, a law might say:

- “Agents cannot modify Ω‑constructors.”  
- “Emergence is limited to UI, not logic.”  
- “Time cannot be reversed.”  
- “Only certain universes can fuse.”

This is all encoded in glyphs and interpreted by the runtime.

---

## 7.4 Law hierarchies

Using ΠMetaGraph (Σ₁₂), you can define:

- Law hierarchies.  
- Law inheritance.  
- Law conflicts and resolutions.

Universes can:

- Share laws.  
- Override laws.  
- Inherit laws.
